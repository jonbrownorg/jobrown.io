ELECTRONIC ARTWORK


ACCEPTABLE MEDIA:
In PC or MAC Formats
- Floppy Disks
- CDs
- Zip Disks
- E-Mail Attachments (Under 2 MB)

NOTE: Never send original disks. Always keep a copy, disks can be damaged in transit.

WHAT TO INCLUDE ON TRANSFER MEDIA:
For Documents Send
- All page layout files.
- All linked image files.
- All fonts used in document.
- Full color hard copy or faxed copy (Please fax a hard copy if order is e-mailed; include separations when faxing).
For Logos Send
- B/W: A good quality B/W print, or a 600-1200 DPI electronic file. Scan original at the anticipated print size or larger.
- Color: A 300 DPI CMYK electronic file. Scan original at the anticipated print size or larger.
- Save scanned images in .tif, .eps, .bmp, .gif, or .jpg formats.


SUPPORTED APPLICATIONS:
Quark Xpress - Up to Version 4.1
- Include all page layout file(s) in .qxd format.
- Include all art support files in .tif or .eps formats.
- Include all printer and screen fonts or TrueType fonts. Be sure to include all styles used (regular, bold, italic, bold italic).
- Include a printed hard copy.

Adobe PageMaker - Up to Version 6.5
- Include all page layout file(s) in .pm format.
- Include all art support files in .tif or .eps formats (Do not embed images).
- Include all printer and screen fonts or TrueType fonts. Be sure to include all styles used (regular, bold, italic, bold italic).
- Include a printed hard copy.

Adobe Illustrator - Up to Version 8.0
- Save as .eps format.
- Included any placed images.
- Change any spot colors to CMYK.
- Create outlines on all text, or include all printer and screen fonts or TrueType fonts. Be sure to include all styles used (regular, bold, italic, bold italic).
- Include a printed hard copy.

Adobe Photoshop - Up to Version 5.0
- Save as .eps or .tif formats.
- Scan original at anticipated output size (300 DPI for color, 600-1200 DPI for B/W bitmap).
- Convert color files to CMYK mode.

Microsoft Word
- Embed your fonts: Open Tools -> Select Options -> Choose Save Tab -> Check Embed TrueType Fonts box.
- Imported images should be at least 300 DPI.
- Not recommended for complex jobs.
