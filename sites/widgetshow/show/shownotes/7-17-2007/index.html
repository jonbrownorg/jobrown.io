Comment on http Request for update checker.
Why use XMLHttpRequest to retrieve the version number? Why not just do
widget.system(”/usr/bin/curl www.jonbrown.org—ntracker.php”,compareVersion);
and save yourself some code?
Very true, that is a better alternative to using HTTP Requests, and I encourage everyone to reduce their code-base by using such an alternative. A better tutorial awaits!
